Blauer Nue font files

Place these WOFF2 files here for the logo and subheading to use Blauer Nue:

  BlauerNue-Light.woff2    (weight 300 – subheading)
  BlauerNue-Regular.woff2  (weight 400)
  BlauerNue-Bold.woff2     (weight 700)
  BlauerNue-ExtraBold.woff2 (weight 800 – logo)

If your files have different names, either rename them to match above or update the @font-face src in index.html.

Blauer Nue is a commercial font (e.g. MyFonts, Envato). Free versions for personal use can be found via search; ensure you have the right license for your use.
